# Odin Flexbox Project

A landing page built as part of [The Odin Project](https://www.theodinproject.com/) Foundations course — Flexbox section.

## What I Learned

- Using `display: flex` to build page layouts
- Controlling alignment with `justify-content` and `align-items`
- Using `flex-direction`, `gap`, and `flex-wrap` for responsive structure
- Nesting flex containers for complex layouts
- Controlling image sizing with `max-width` and `object-fit`
